================================================================================
                           Smart Contract Details
================================================================================

State-changing functions:
  1. createCampaign(title, description, targetAmount, durationDays)
     - Create a new campaign
     - Emits CampaignCreated

  2. contribute(campaignId)
     - Contribute funds (payable)
     - Emits ContributionMade

  3. withdrawFunds(campaignId)
     - Withdraw funds (creator only)
     - Emits FundsWithdrawn

View functions:
  1. getCampaignDetails(campaignId) - returns campaign details
  2. getContributorAmount(campaignId, contributor) - returns contribution amount

Events:
  1. CampaignCreated(campaignId, creator, title, targetAmount, deadline)
  2. ContributionMade(campaignId, contributor, amount)
  3. FundsWithdrawn(campaignId, creator, amount)

Security:
  ‚úÖ Access control (onlyCreator)
  ‚úÖ Input validation (require checks)
  ‚úÖ State checks (deadline, target)
  ‚úÖ Safe fund transfer (call pattern)
